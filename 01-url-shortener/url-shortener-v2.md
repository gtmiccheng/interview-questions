# URL Shortener API Service v2

## Requirements

- Refer to the V2 [OpenAPI specification](openapi-v2.yml) (https://go.gov.sg/swe-ar-interview-question) ([Swagger](https://raw.githubusercontent.com/gtmiccheng/interview-questions/refs/heads/main/01-url-shortener/openapi-v2.yml)) for the updated request/response formats.
- We will want to support a custom slug for the shortened URL. 
- If the user provides a custom alias, the service should use that instead of generating a random one.
